Donate using an Amazon Gift Card

Amazon eGift Cards are a great way to annonymously gift someone.

  1. Goto Amazon and navigate to eGift Cards.
  2. Select an eGift Voucher of any value or style.
  3. Fill in the details, select the delivery type Email and the recipient jay@protonmail.com. You can put anything in the name and message field. You can leave the Delivery Date as Now.
